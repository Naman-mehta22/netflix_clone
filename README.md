# 🎬 Netflix Clone

A Netflix-inspired landing page clone built using **HTML5** and **CSS3**. This project recreates the Netflix homepage design with a responsive layout and modern UI styling. The purpose of this project is to practice frontend development concepts and improve web design skills.

---

## 🚀 Features

- 🎥 Netflix-inspired UI
- 📱 Responsive webpage design
- 🎨 Modern styling and layout
- 🖼️ Background images and media sections
- ▶️ Embedded video section
- ✨ Hover effects and clean design
- 📋 FAQ section

---

## 🛠️ Technologies Used

- HTML5
- CSS3

## ⚙️ How to Run the Project

1. Clone the repository:

```bash
git clone https://github.com/yourusername/netflix-clone.git
```

2. Open project folder:

```bash
cd netflix-clone
```

3. Run the project:

Open `index.html` in your browser.
